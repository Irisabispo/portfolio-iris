- Layout responsivo

- Animações suaves

##  Como executar

Clone o repositório:

```bash
git clone https://github.com/Irisabispo/portfolio-iris.git
```

Entre na pasta:

```bash
cd portfolio-iris
```

Abra o arquivo `index.html` no navegador ou utilize uma extensão como **Live Server** no VS Code.
